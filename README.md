# Mongoid::Preferences

Add preferences hash to Mongoid model.

## Installation

Add this line to your application's Gemfile:

    gem 'mongoid-preferences'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongoid-preferences
